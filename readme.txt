CallMeBot setup:
1) Add CallMeBot WhatsApp number to your contacts (see https://www.callmebot.com/)
2) Send: "I allow callmebot to send me messages"
3) Wait for the reply containing your APIKEY
4) Put that APIKEY into main.py

Important: do not commit passwords or API keys to public repos.
